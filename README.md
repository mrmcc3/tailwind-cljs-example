### Example ClojureScript + Tailwind Setup

This example uses:

* [Shadow CLJS](http://shadow-cljs.org/) for ClojureScript
* Tailwind + PurgeCSS + CleanCSS for CSS 
* [Reagent](https://github.com/reagent-project/reagent) for React

For development

``` 
$ yarn dev
```

For release

``` 
$ yarn build
```
